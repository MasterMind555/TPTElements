About Powder Toy Element Coder

It started off as something I created with enthusiasm back in summer 2012. I had overthought the entire thing and used loads of unnecessary techniques (such as PHP, AJAX and Server-side Includes) when creating it. The rewrite uses only three files and technologies and it looks minimalistic. I like minimalistic. The colour scheme has been worked through with colorschemedesigner.com. The text scheme is here: #3u21TbFuVw0w0 and the background is a single color anyways. The code provided is from The Powder Toy by various developers, hosted on GitHub.
